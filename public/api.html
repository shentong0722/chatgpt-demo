ST API 服务使用说明

  1. 如何使用：直接将客户端BASE_URL改成api.st0722.top即可，有些客户端（比如沉浸式翻译）可能需要输入完整路径https://api.st0722.top/v1/chat/completions
  2. 密钥获取：关注微信公众号“ST是只小羊吖”，后台回复API即可免费获取（后期可能改为收费）
  3. 模型对齐：目前API请求的模型可选项有很多种类，对小白并不友好，而且有些客户端并不会及时更新到最新的模型，而我们的API代理服务会自动帮您对齐到最新的模型
  4. 提示词注入：大多数用户并不知道如何撰写提示词，而且有些客户端无法自定义提示词或者修改麻烦，我们的API代理会自动在请求中注入我们撰写的最佳提示词
